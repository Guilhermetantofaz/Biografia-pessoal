Imagens

O comando para inserir uma imagem é: 'img src=""'. Tem como buscar a imagem direto de um link, ou pode adicionar a imagem dentro do projeto.

O método do link é simples, basta copiar o link da imagem e colar dentro das aspas depois do 'src='.
Agora para adicionar a imagem no projeto, será necessário baixar a imagem e depois criar uma pasta, dentro da pasta do projeto, e colocar a imagem nessa pasta. Para buscar a imagem você terá que digitar o nome da pasta e depois do arquivo da imagem. Por exemplo, se o nome da pasta for "img" e do arquivo "Imagem 1", eu teria que escrever img/Imagem 1.jpeg.

Imagens através do link, costumam demorar menos para carregar. Tem também o comando alt="", que serve para descrever a imagem, será útil para caso um deficiente visual entre no site, e ajuda a melhorar os fatores de busca.

Para mudar o tamanho da imagem, deve-se utilizar o comando 'width', e para ter uma idéia de tamanho, basta 'inspecionar' a página e clicar na imagem.

Enviar para outra página/site.

Para isso é necessário usar o comando 'a href=""' e escrever o caminho de referência, por exemplo: "contato/contato.html", o comando 'a href=""', não se fecha sozinho.

Entre em contato

Erros: esqueci do ".html" na hora de buscar uma imagem; esqueci que o comando 'a' tem que ser fechado; para aninhar as listas, o comando 'ul' ou 'ol', devem estar dentro do 'li'; 'strong' deve ser menos usado, o 'em' que serve para enfatizar.
